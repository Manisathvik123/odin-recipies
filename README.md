# odin-recipies

A recipe collection built as my first HTML project through The Odin Project.
No CSS, no JavaScript — just pure, unstyled, brutally honest HTML.

## What's Cooking

- 🍔 Turkey Burger
- 🥗 Mashed Potato Salad
- 🌮 Shrimp Tacos
- 🧁 Cupcakes

## About

This is project one of many. It looks rough — that's the point.
The focus here was structure and linking, not looks.
Styling comes later, and when it does, this page will never be the same.
